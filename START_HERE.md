## 🚨 Before You Run

### Step 1: Set Up Database (One-Time Only)
1. Go to https://supabase.com/dashboard
2. Select project: `pgwcfazhwuzxcqbqbjat`
3. Go to **SQL Editor > New Query**
4. Open `supabase_schema.sql` in this project
5. Copy all the SQL
6. Paste into Supabase SQL editor
7. Click **Run**
8. Wait for completion (should show no errors)

### Step 2: Run the App
```bash
npm install
npm run dev
```
